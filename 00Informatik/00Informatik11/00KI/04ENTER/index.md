# Automatisches Erstellen von Entscheidungsbäumen mit ENTER

1. Lade die Datensätze herunter:
https://kex.fim.uni-passau.de/ENTER_Online/html/BeschreibungFunktionen.html

2. Erstelle auf dieser Seite den Entscheidungsbaum
https://kex.fim.uni-passau.de/ENTER_Online/html/ENTER_Online.html  
Verwende hierzu die Datei [Trainingsdaten_Fische.csv](Trainingsdaten_Fische.csv)

3.  Teste den Baum
Verwende hierzu die Datei [Testdaten_Fische.csv](Testdaten_Fische.csv)

4.  Analysiere die Testergebnisse in der Konfusionsmatrix
Berechne die 
- Genauigkeit G = (TP+TN)/(TP+TN+FP+FN)
- Sensitivität S = TP/(TP+FN)
- Präzision: P = TP/(TP+FP)

5. Erstelle einen weiteren Baum mit dem   [Großen_Datensatz](Großer_Datensatz_Fische.csv)
6. Analysiere diesen Baum
