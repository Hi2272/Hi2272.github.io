## Defintion: 
Arthur Samuel defined machine learning in 1959 as the "field of study that gives computers the ability to learn without being explicitly programmed".

## Einstiegsbeispiel
`````C++
if (mouseX>200){
    print("Right side");
} else {
    print("Left side");
}
`````

Klassisches Programm, das erkennt, ob die Maus rechts oder links im Fenster ist.  

ML-Ansatz: 
Datensatz mit gelabelten X-Koordinaten:  
100: Left  
300: Right  
40: Left  
230: Right  
Das ML-System soll aus diesem Datensatz lernen, wann eine Koordinate links oder rechts steht.
