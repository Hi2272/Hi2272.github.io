## Allgemeines
Verwendung von pre-trained-models zur Objekterkennung:  

Modell: MobileNet:  
- Von Google erstellte Modelle zur Objekterkennung.
- Optimiert für den Einsatz auf Smartphones.
- Modell wird in den Speicher geladen und die Erkennung erfolgt auf dem Gerät. Die Daten werden nicht auf einen Server geschickt.
- Trainiert auf einem Teil von Imagenet:

[ImageNet](https://www.image-net.org/): Datenbank von Bildern, auf der MobileNet trainiert wurde.
