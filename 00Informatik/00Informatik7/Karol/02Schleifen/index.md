# Kurzer Code durch Schleifen
### Code verkürzen
Mit 20 Codeblöcken baut Karol einen Rahmen um seine Welt.
Das geht aber auch kürzer.
Verwende Schleifen mit fester Wiederholungszahl, so dass 5 Blöcke ausreichen.
