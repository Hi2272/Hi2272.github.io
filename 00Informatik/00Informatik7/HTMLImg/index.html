Bilder in HTML

Bilder werden mit dem <img>-tag eingefügt.
Direkt hinter dem img können Attribute in der Form <img attributname="wert"> stehen.
  1. Das Attribut src="Dateiname" gibt die Quelle (engl. source) des Bildes an.
    Füge das Bild "Palme.jpg" im Editor ein.
  2. Lege über das zusätzlich Attribut width="100px"; die Breite auf 100 Pixel fest.
  3. Lege über das Attribut height="50px"; die Höhe auf 50 Pixel fest.
