Esp Home ist ein Add-On zu Home Assistant, mit dem ESP32-Module in ein Home Assistant System eingebunden werden können.   
Die angeschlossenen Sensoren oder Aktoren werden in Yaml-Dateien konfiguriert. Die Software kompiliert mit diesen Daten Code und flasht diesen auf den über USB angeschlossenen Mikrocontroller.
