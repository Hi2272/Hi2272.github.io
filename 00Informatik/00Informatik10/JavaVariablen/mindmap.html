Variablen in Java

1. Definition

  • Speicherplatz für Daten
  • Benannte Referenzen auf Werte

2. Arten von Variablen

  • Lokale Variablen
    • Innerhalb von Methoden oder Blöcken
    • Sichtbarkeit beschränkt auf den Block
  • Instanzvariablen
    • Zugehörig zu einem Objekt
    • Lebensdauer entspricht der Lebensdauer des Objekts
  • Klassenvariablen (statische Variablen)
    • Zugehörig zur Klasse, nicht zu einem Objekt
    • Mit dem Schlüsselwort static deklariert
    • Geteilt von allen Instanzen der Klasse

3. Datentypen

  • Primitive Datentypen
    • int - Ganzzahlen
    • double - Gleitkommazahlen
    • char - Einzelne Zeichen
    • boolean - Wahrheitswerte
  • Referenzdatentypen
    • Objekte
    • Arrays
    • Strings

4. Deklaration und Initialisierung

  • Deklaration
    • Beispiel: int zahl;
  • Initialisierung
    • Beispiel: zahl = 5;
  • Kombination
    • Beispiel: int zahl = 5;

5. Sichtbarkeit und Zugriffsmodifizierer

  • Öffentlich (public)
  • Privat (private)
  • Geschützt (protected)
  • Standard (package-private)

6. Konstante Variablen

  • Verwendung von final
  • Beispiel: final int MAX_WERT = 100;

7. Typkonvertierung

  • Automatische Typkonvertierung (Widening)
    • Beispiel: int zu double
  • Manuelle Typkonvertierung (Narrowing)
    • Beispiel: double zu int

8. Gültigkeitsbereich (Scope)

  • Methoden-Scope
    • Parameter
  • Klassenscope
    • Attribute
  • Block-Scope
    • lokale Variablen

9. Best Practices

  • aussagekräftige Namen wählen
  • Variablen möglichst nah am Einsatzort deklarieren
  • Verwendung von Konstanten für feststehende Werte
  • Vermeidung von globalen Variablen

10. Beispiele

  • Einfache Deklaration: java int zahl = 10;

  • Verwendung von Instanzvariablen: java public class Beispiel { int instanzVariable; }

  • Verwendung von Klassenvariablen: java public class Beispiel { static int klassenVariable; }
